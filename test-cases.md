# Test Cases Documentation

## Test Case 1: Login with Valid Credentials
- **Description**: Verify that a user can log in with valid credentials.
- **Covered Flow**: Login functionality.
- **Steps**:
  1. Navigate to the login page.
  2. Enter valid email and password.
  3. Click the "Iniciar sesión" button.
- **Expected Result**: The user is redirected to the "Dishes" page.

## Test Case 2: Add a New Dish
- **Description**: Verify that a user can add a new dish.
- **Covered Flow**: Dish creation functionality.
- **Steps**:
  1. Navigate to the "Dishes" page.
  2. Click the "+ Agregar Platillo" button.
  3. Fill in the dish details (name, description, preparation time, etc.).
  4. Click the "Guardar" button.
- **Expected Result**: The new dish appears in the list on the "Dishes" page.

## Test Case 3: Edit an Existing Dish
- **Description**: Verify that a user can edit an existing dish.
- **Covered Flow**: Dish editing functionality.
- **Steps**:
  1. Navigate to the "Dishes" page.
  2. Click the "Editar" button for a specific dish.
  3. Modify the dish details.
  4. Click the "Guardar" button.
- **Expected Result**: The updated dish details are reflected on the "Dishes" page.

## Test Case 4: Delete a Dish
- **Description**: Verify that a user can delete a dish.
- **Covered Flow**: Dish deletion functionality.
- **Steps**:
  1. Navigate to the "Dishes" page.
  2. Click the "Eliminar" button for a specific dish.
- **Expected Result**: The dish is removed from the list on the "Dishes" page.
